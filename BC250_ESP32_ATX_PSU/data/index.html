Detection and spoofing options
Controller detection methodsiThese options control both controller discovery and background detection used to turn on the PC.

Bluetooth Low Energy (BLE) mode listens for BLE advertisements.
Bluetooth Classic (pairing mode) runs Classic inquiries while controllers are discoverable.
Bluetooth Classic (paired to the spoofed address) listens for reconnect attempts from controllers paired with the configured spoof address.

Disabling unused methods can reduce radio contention and improve detection time. Changes are saved immediately.
Choose how the ESP32 discovers and monitors controllers
Changes are saved and applied immediately.
Bluetooth spoof addressiSet the MAC address of the BC250 Bluetooth adapter that the gamepads are already paired with. While the PC is off, the ESP32 uses it to notice a paired gamepad trying to reconnect.

Leave this empty to disable spoofing.
This is not required for BLE gamepads.

You can get the adapter MAC address with the `bluetoothctl list` command.

A restart is required after changing it.
